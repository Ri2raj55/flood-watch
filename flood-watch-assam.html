Ideas to build on this
WhatsApp botLet people submit a report by sending a photo + location pin to a WhatsApp number — no app needed, works on low-end phones.
Auto severity from photoRun submitted photos through a vision model to flag likely severity, so the map self-corrects if someone under- or over-reports.
Authority hand-offAuto-email or SMS the district disaster management office when reports cluster in one ward within a short window.
Heatmap viewPlot reports on an actual map (Leaflet + OpenStreetMap) instead of a card grid, so people can see flooding spread street by street.
SMS fallbackFor areas with patchy data, a short SMS code (e.g. "FLOOD ANIL NAGAR KNEE") logged the same way as the form.
Verified badgeLet a few trusted local volunteers mark reports as confirmed, so the feed stays reliable during panic spikes.
